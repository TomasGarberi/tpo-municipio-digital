# TPO - Ingeniería de Datos II
## Tema 12 - Municipio Digital

Trabajo práctico integrador de Ingeniería de Datos II.

## Motores de la primera entrega
- MongoDB
- Neo4j

## Estructura
- docs/: informe, diagramas y evidencias
- mongo/: scripts, datos y resultados de MongoDB
- neo4j/: scripts, datos y resultados de Neo4j
- shared/: decisiones de modelado y dataset común

## Objetivo
Resolver la primera entrega implementando y documentando MongoDB y Neo4j de forma independiente, con datos coherentes entre ambos motores.
